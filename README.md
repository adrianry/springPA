# Camunda BPM Process Application

Diese Applikation soll die verschiedenen Möglichkeiten von BPMN und der Interpretation durch die Engine zeigen.
Es wird die shared Process Engine verwendet. Diese steht als JBoss Modul in der JBoss AS 7 Distribution von Camunda.org zur Verfügung.
CamundaBpmProcessApplication.java weist die Engine an eine Process Application zu registrieren.


This project has been generated by the Maven archetype
[camunda-archetype-servlet-war-7.3.2](http://docs.camunda.org/latest/guides/user-guide/#process-applications-maven-project-templates-archetypes).
